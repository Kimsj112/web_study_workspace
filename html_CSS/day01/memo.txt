CSS(Cascading Style sheets)
    HTML로 만든 웹 문서의 스타일(디자인)을 담당한다.
    글꼴, 색상, 테두리, 정렬, 배치(레이아웃) 방법등의 외관을 설정하는 것이다.

CSS 형식
    속성1 : 속성값1;
    속성2 : 속성값1;

스타일 적용 방법
    1. 인라인 스타일
      스타일을 적용시킬 택에 Style 속성을 사용한다.
      
      인라인 스타일은 간단하게 사용하기에는 좋으나 코드가 길어질수록 가독성이 떨어지며,
      css와 html이 뒤섞여 유지보수가 매우 힘들어진다.

    2. 내부 스타일
      <head> 영역에 <Style></Style> 태그를 사용하여 각 요소들의 스타일을 정의한다.

      형식은 다음과 같다.
      선택자 {
        속성1 : 속성값1;
        속성2 : 속성값2;
      }

      선택자 : 어떤 요소를 선택하기 위해 사용하는 것

    3. 외부 스타일
      별도의 css파일을 만들어 적용시킨다.
      css와 html을 완전히 분리하여 관리할 수 있기 때문에 가장 많이 사용하는 방식이다.
      css파일을 html파일에 적용시키기 위해서는 <head>영역에 <link>태그를 사용한다.
      <link rel="Stylesheet" href="파일경로">

      rel 속성은 현재 페이지와의 관계를 의미하며, href는 적용시킬 css파일 경로를 작성한다.
      



id와 class
    id와 class 속성은 특정 태그들을 식별하고 그룹화 하는데 사용된다.
    모든 태그에 사용가능한 속성이다.

    - id
    특정 태그에 식별 가능한 고유값을 부여함
    선택자로 사용할 때는 #id로 사용한다.

    - class
    여러 태그를 하나의 집단으로 묶어줌
    선택자로 사용할 때는 .class 로 사용한다.

    하나의 태그에 id와 class를 동시에 가질 수 있으며 class를 여러개 가질수도 있다.
    주로 class를 많이 사용하며 id는 그 페이지에서 중요하고 유일한 요소에만 부여하는ㄴ 것이 좋다.
