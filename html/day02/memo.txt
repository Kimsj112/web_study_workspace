태그의 종류
    <strong></strong>, <b></b> : 굵은 글씨
    <br> : 줄바꿈 태그
    <hr> : 가로선을 그려줌

    * 닫는 태그가 없는 것을 '빈 태그'라고 부른다.

- 리스트 태그
    <ol></ol> : Ordered List의 약자, 순서있는 리스트를 만든다.
    <ul></ul> : Unordered List의 약자, 순서없는 리스트를 만든다.
    <li></li> : List의 약자, ol 또는 ul 태그의 자식으로 사용한다.

- 테이블 태그(표 만들기)
    <table></table> : 표를 만드는 태그
    표의 제목, 행의 수, 열의 수 등을 다른 태그를 이용해 표현한다.

    <tr></tr> : 행을 만드는 태그
    <th></th> : 제목행에 칼럼을 만드는 태그`
    <td></td> : 칼럼을 만드는 태그
    <caption></caption> : 표의 제목

- 이미지 삽입 태그
    <img src = "파일경로" alt="설명" />
    src 속성은 넣고 싶은 이미지의 경로를 입력하며, 웹상의 주소를 입력해도 된다.
    alt는 이미지에 대한 설명이며 만약 어떠한 이유로 이미지가 뜨지 않으면 설명이 대체되어 나온다.

- 링크를 거는 a태그 (앵커 태그)
    <a href="링크주소 | 경로"> 링크를 걸어줄 요소</a>
    a태그로 감싼 요소는 해당 경로로 링크가 걸린다.


인라인/블록 요소
    <span></span> : 대표적인 인라인 태그
    <div></div> : 대표적인 블록 태그

- span과 div태그는 사이트 화면의 레이아웃을 나누는데 주로 사용되며 매우 중요하다.
- 둘 다 의미가 없다. (시멘틱하지 않은 태그)

인라인 요소와 블록 요소
    요소를 여러 관점으로 분류할 수 있다.
    그 중 가장 중요하게 봐야할 것은 인라인과 블록이다.
    HTML의 모든 요소는 인라인과 블록으로 나뉜다.

    인라인 요소
    - 내용물(컨텐츠)의 크기만큼만 영역을 차지하므로 다른 요소와 같은 줄에 위치할 수 있다.
    즉, 줄 바꿈이 되지 않는다.
    - 대표적으로 span, strong, b, i, em 등이 있다.

블록 요소
    - 내용물과 상관없이 한 줄을 다 차지하므로 다른 요소와 같은 줄에 위치할 수 없다.
    즉, 너비 속성이 100%를 다 차지하고 자동으로 줄 바꿈 된다.

    - 대표적으로 div, h1, p 등이 있다.
